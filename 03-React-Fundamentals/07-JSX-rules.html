JSX Rules

  • Always always always return single element (one parent element)
    • use HTML semantics elements - section/article
    • Fragments - allows us to group elements without adding extra nodes
      • <React.Fragment>...rest of the return </React.Fragment>
      • <>...rest of the return </>
  • camelCase naming convention for HTML element attributes
    • htmlFor = " "
    • onClick = " "
    • tabIndex = " "
    • readOnly = " "
  • use className instead of class
  • close every element ()
    • <img />
    • <input />
  • closing parenthesis return () - Not a must but make sure that if you are not using () then the opening tags MUST be within the same line with return keyword
