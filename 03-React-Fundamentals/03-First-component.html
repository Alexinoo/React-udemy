First Component

  • In order to create a component in react, essentially we need to create a function
  • Always Always Always Always think of component as a function
  • Anytime you want to create a component , think of I need to create a function
  • Can be a function with good old function keyword or create a component with arrow function
  • Whether to use ( arrow function OR good old function keyword ) depends on your preference
  • Component rules
    • Start with a capital letter
    • must! must! must! must! must! return some kind of html element ( Technically called JSX ) from the function
    • enclosed with closing tags
  • Once we are done creating a component then we need to export it either as (default or named) export so that we can use it anywhere within our application
