What is React ?

  • A javascript library for building user interfaces
  • Features :
    • Was built and still maintained by Facebook (Meta)
    • Initial release was in 2013 and is currently by far most popular JS library to build user interfaces
    • Some of the React competitors are : Angular , Vue , Svelte
    • All about components - Independent chunk of user interfaces - can be as small as 1 HTML element e.g. Button or h2 element
    • You can build a bunch of independent , isolated and most importantly re-usable UI and build super big complex apps without going insane
    • Your app can have as many components as you would like but will always have at least one root component
    • Component code is stored in one place and therefore changes are only done in one place and all the instances gets updated automatically
    • Speed - React uses Virtual DOM behind the scene where only the component that needs to be updated is affected
      • What is even cool is that it is done without re-rendering the whole app
